oneccl_bind_pt: Overview & Setup

This package is part of an ongoing effort to abstract backend-level collective operations with compatibility for evolving distributed training frameworks.

The goal is to provide experimental bindings that can potentially adapt to hardware-layer messaging systems in a plug-and-play manner.

Documentation is currently in draft status. Implementation details and platform-specific configuration are subject to change in future iterations.
